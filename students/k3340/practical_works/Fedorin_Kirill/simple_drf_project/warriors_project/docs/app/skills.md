# Skills Endpoint

## Получение списка умений

- **URL:** `/war/skills/`
- **Метод:** `GET`
- **Описание:** Возвращает список всех умений.

## Создание умения

- **URL:** `/war/skills/create`
- **Метод:** `POST`
- **Описание:** Создает новое умение.
- **Параметры:**
  - `title` (string): Название умения.
